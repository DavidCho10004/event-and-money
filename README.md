## 프로젝트 개요

주요 사건(전쟁, 금융위기, 팬데믹, 정책 변동 등) 발생 시 자산군별(주식, 원유, 금, 환율, 채권 등) 가격 반응을 **단기(D+1, D+7) → 중기(D+30) → 장기(D+180, D+365)** 시간축으로 정량 분석하고, **파급 체인(1차/2차/3차 변인)**을 시각화하는 프로젝트.

### 핵심 차별점
- 기존 책/리포트: "S&P 500이 평균 5% 하락 후 3주 만에 회복" 수준
- **Event & Money**: 사건 유형별 × 자산군별 × 시간축별 × 파급 단계별 매트릭스
- 실전 투자(1억 원 운용) 기록을 병행하여 이론+실전을 결합

| 구성요소 | 기술 | 비고 |

## 빠른 시작

```bash
# 1. DB 시작
docker-compose up -d

# 2. 환경변수 설정
cp .env.example .env

# 3. 백엔드
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000

# 4. 프론트엔드 (Phase 2)
cd frontend
npm install
npm run dev
```
